Le code César

Description

Le code César est une méthode de chiffrement très simple utilisée par Jules César dans ses correspondances secrètes. Le texte chiffré s'obtient en remplaçant chaque lettre du texte clair original par une lettre à distance fixe, toujours du même côté, dans l'ordre de l'alphabet. Pour les dernières lettres (dans le cas d'un décalage à droite), on reprend au début. Par exemple avec un décalage de 3 vers la droite, A est remplacé par D, B devient E, et ainsi jusqu'à W qui devient Z, puis X devient A etc.

Aide à l'encodage


Astuces

On peut décider ou non de fournir la clé de décriptage en fonction de la difficulté que l'on veut donner à notre énigme:

Si le message est assez long, quelqu'un d'attentif pourra repérer des enchainement connus (une lettre seule est un A ou un Y, la lettre la plus présente est souvent un E, etc) sans qu'on lui donne le chiffre de César.

Si vous choisissez de donner le décallage, vous pouvez le faire de façon brut, ou bien déguisé en donnant par exemple le mot "Arranger" pour A rend G.
